# Monetize

Monetize is the newest version of the old smatched.io website, rebuild with React.js

## Installation

In order to install and run a local copy, run the following commands:

```bash
pnpm install
```

```bash
pnpm start
```

If you don't have pnpm, install it via:

```bash
npm i -g pnpm
```
